## The Road to Linux - 个人博客系统运维自动化部署项目

### 项目简介

​		本项目基于**LNMP+Docker**技术栈，实现个人博客系统（WordPress）的**容器化一键部署**，并开发自动化运维脚本完成服务器健康监控、MySQL 数据库定时备份，同时实现日志管理、定时任务调度等企业级运维基础功能。

​		项目覆盖 Linux 系统管理、Shell 脚本开发、Docker 容器编排、Nginx 配置、MySQL 运维等核心技能，是运维入门的实战型项目，所有操作均贴合企业日常运维场景。

### 技术栈

- **操作系统**：Ubuntu 22.04
- **容器化**：Docker + Docker Compose 3.8
- **服务端**：Nginx (alpine)、MySQL 8.0、WordPress (PHP7.4-FPM)
- **运维脚本**：Shell
- **任务调度**：Linux Crontab
- **其他**：Git（版本管理）、Netstat/SS（网络排查）

## 项目功能

1. 🚀 **一键容器化部署**：通过 Docker Compose 快速启动 Nginx+MySQL+WordPress 整套博客环境，实现服务解耦与持久化存储
2. 🔍 **服务器健康监控**：自动化检测 CPU、内存、磁盘、关键端口（80/3306）状态，输出检测日志，快速定位服务异常
3. 📦 **MySQL 定时备份**：实现数据库全量备份、压缩存储、7 天历史备份自动清理，防止数据丢失
4. 📜 **日志管理**：分离脚本运行日志、Nginx 服务日志，方便问题排查与日志归档
5. ⏰ **定时任务自动化**：通过 Crontab 配置监控、备份脚本的定时执行，实现无人值守运维
6. ⚡ **静态资源优化**：Nginx 配置静态文件缓存，提升博客访问速度
